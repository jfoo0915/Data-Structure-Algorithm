Member Maintenance Module: 
Press '2' to proceed to the member maintenance module; 
Enter '1000' for admin id; 
Enter '1111' for admin password;

Add new member: 
Press '1' to proceed to the add new member; 
type any string value to input the name (not allow name duplication in the system eg: Yang);
type 'M' or 'F' to input gender; 
type any number to input contact no; 
type '1001'-'1008' number to input room id; 
type any int value to input the password;
